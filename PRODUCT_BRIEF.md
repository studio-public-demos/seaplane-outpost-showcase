# Barbie Dress-Up Party Game

## Product Definition
An interactive 3D web-based dress-up game where school kids can decorate and style a 3D Barbie figure for a party. Built with Three.js for real-time 3D rendering in the browser.

## Target Users
School-age children (ages 5-12) who enjoy fashion, dress-up, and creative play.

## Core Features
1. **3D Barbie Model** - Procedurally generated Barbie figure with adjustable features
2. **Dress Styles** - Multiple outfit options (Ball Gown, Party Dress, Princess Dress, Casual)
3. **Color Customization** - Dress, hair, skin, and shoe colors with preset swatches
4. **Accessories** - Crown/tiara, necklace, bracelet, earrings, handbag (toggle on/off)
5. **Scene Backgrounds** - Different party-themed backgrounds
6. **Party Celebration** - Confetti animation when "Party Ready!" is clicked
7. **Responsive Design** - Works on desktop, tablet, and mobile devices

## Technical Stack
- Three.js (CDN) for 3D rendering
- Vanilla HTML/CSS/JavaScript (no framework needed)
- Single self-contained HTML file
- Mobile-first responsive design

## User Journeys
1. Open the game → See a 3D Barbie on a podium
2. Tap dress style → Choose from 4 styles
3. Tap colors → Pick dress/hair/skin/shoe colors
4. Tap accessories → Toggle crown, necklace, bracelet, earrings, handbag
5. Tap background → Choose party scene
6. Tap "Party Ready!" → See confetti celebration
7. Rotate and zoom the Barbie using touch/mouse

## Non-Goals
- No user accounts or save functionality (v1)
- No sound/music (v1)
- No photo upload or sharing
