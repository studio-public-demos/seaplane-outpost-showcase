# Seaplane Outpost Showcase

Procedural Three.js seaplane outpost featuring custom GLSL Gerstner water, floatplane buoyancy, physical wakes and spray, dynamic lighting and weather, animated workers, and simulated flight operations.

## Features

- **Gerstner Water**: Custom GLSL shader with vertex displacement, foam crests, and depth-based coloring.
- **Procedural Seaplane**: Fully procedural model with buoyancy physics.
- **Interactive Dashboard**: Control panel for flight operations (Takeoff/Land) and real-time environment telemetry.
- **Dynamic Weather**: Real-time transitions between clear, overcast, and rainy conditions.
- **Animated Outpost**: Workers, circling seabirds, and procedural island geometry.

## Built with

- [Three.js](https://threejs.org/)
- [NebulaCloud Studio](https://nebulacloud.studio/)

## How to run

Simply open `seaplane_outpost.html` in a modern web browser.
